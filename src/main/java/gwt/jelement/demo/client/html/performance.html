Performance API Demo

The Performance API is available on all modern browsers and IE 11. On Chrome it additionally provides memory information. The performance timing display code was intentionally written to demonstrate accessing properties through an object's property descriptors. This code was also intentionally written using two methods not supported in IE 11 (Object.entries and Object.getOwnPropertyDescriptors, to demonstrate two ways of creating polyfills with gwt-jelement.
